# Netty







## 说一说Netty如何建立长连接





1. 创建连接和通道
2. 发心跳包告知连接状态。
3. 发消息
4. 一段时间没收到心跳包或者用户主动关闭之后关闭连接





## 说一说Netty有什么特点



1. 高性能的NIO网络通信框架
2. 封装了NIO简化了开发。
3. 可配置IO线程数
4. 使用了volitale CAS算法，并发类来保证线程安全。



## 说一说Netty的线程模型



1.netty用了两个线程池，boss线程池和work线程池，boss线程池的线程负责接收数据，work线程池的线程负责IO操作。

2.netty的Reactor模型有单线程模型和多线程模型，单线程模型的数据接受操作和IO操作都通过一个线程来完成，高并发环境不合适。多线程模型中，boss线程池的线程负责建立连接接受数据，work线程池中的线程负责IO操作。
